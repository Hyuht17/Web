# Bài tập thực hành Tuần 6 - Môn Công nghệ Web

## Mục tiêu
- Thực hành HTML5 semantic elements
- Làm việc với CSS Grid và Flexbox
- Tạo responsive design
- Sử dụng JavaScript ES6+ features
- Tương tác với DOM API

## Cấu trúc bài tập
1. **Bài 1**: HTML5 Semantic Layout
2. **Bài 2**: CSS Grid Layout
3. **Bài 3**: CSS Flexbox
4. **Bài 4**: Responsive Design
5. **Bài 5**: JavaScript ES6+ Features
6. **Bài 6**: DOM Manipulation
7. **Bài 7**: Form Validation
8. **Bài 8**: Local Storage

## Hướng dẫn làm bài
- Mỗi bài tập có file HTML, CSS, và JavaScript riêng
- Đọc kỹ yêu cầu trước khi làm
- Test kỹ trên nhiều trình duyệt khác nhau
- Sử dụng Developer Tools để debug

## Yêu cầu nộp bài
- Tạo thư mục riêng cho mỗi bài tập
- Đặt tên file theo format: `bai1.html`, `bai1.css`, `bai1.js`
- Nộp đầy đủ source code và screenshot kết quả
